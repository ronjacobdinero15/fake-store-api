# Fake Store API

A fake ecommerce app designed to practice the use of Context API + React Router.

## What I Practiced

- **Custom API**: Honing experience in integrating and interacting with a custom API for fetching and managing product data.
- **React Router**: Mastering the use of React Router for implementing dynamic routing, nested routes, and handling navigation within the application.
  
Feel free to dive into the codebase to see these concepts applied in practice!
